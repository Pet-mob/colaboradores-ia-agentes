# Demanda #27: Layout - Ajustar Posicionamento das Categorias

**Projeto:** Pet.ON.App  
**Tipo:** Layout/Design  
**Status:** Doing  
**Tags:** app  
**Prioridade:** Média

---

## 📋 Descrição

Ajustar o posicionamento visual e styling das categorias de serviços no aplicativo. As categorias podem estar sobrepostas, desalinhadas ou com tamanho inconsistente. É necessário revisar o layout e garantir consistência e responsividade.

### Objetivo
- Melhorar visual das categorias
- Garantir responsividade
- Alinhar com design system
- Melhorar UX de navegação

---

## 🎯 Critérios de Aceitação

- [ ] Categorias estão bem espaçadas (não sobrepostas)
- [ ] Alinhamento consistente (esquerda/centro/direita)
- [ ] Tamanho consistente entre categorias
- [ ] Padding/Margin uniform
- [ ] Responsive em diferentes tamanhos de tela
- [ ] Compatível com iOS e Android
- [ ] Ícones e textos alinhados
- [ ] Sem overflow em telas pequenas
- [ ] Visual aprovado por design

---

## 💻 Detalhes Técnicos

### Componentes Afetados
- Componente de Categorias
- Tela de Busca/Home (`screens/Buscar.js`)
- Stylesheet/Theme

### Possíveis Correções
- Ajustar flexbox layout (flex-direction, justify-content, align-items)
- Revisar padding/margin
- Considerar FlatList para categorias
- Ajustar font-size e dimensões de ícones
- Testar em múltiplas resoluções

### Breakpoints Esperados
- iPhone SE: 375px
- iPhone 8: 375px
- iPhone 12: 390px
- Android diversos: 360px - 540px
