## 📖 Template de Demanda

Cada arquivo segue este template:

```
# Demanda: [Título]
- Descrição do que fazer
- Objetivos claros
- Requisitos funcionais (Backend/Frontend)
- Critérios de aceitação
- Arquivos afetados
- Dependências
- Estimativa de complexidade
- Seções para Implementação e QA
```

## 🎓 Como Usar Este Índice

1. **Selecione por tipo** (Feature, Bug, Ajuste)
2. **Escolha por complexidade** (comece com baixa!)
3. **Verifique as dependências** (execute na ordem correta)
4. **Abra o arquivo .md** completo
5. **Siga as especificações** passo a passo
6. **Atualize o status** conforme progride
